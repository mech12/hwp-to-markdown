## 이슈 및 메모

### 해결된 이슈

1. **markdownify strip 옵션 문제**
   - 문제: `strip=["script", "style"]` 옵션이 태그 내용을 제거하지 못함
   - 해결: BeautifulSoup으로 먼저 해당 태그를 `decompose()`로 완전 제거

### 의존성

- pyhwp>=0.1b12: HWP → HTML 변환 (hwp5html CLI)
- markdownify>=0.11.0: HTML → Markdown 변환
- beautifulsoup4: HTML 전처리 (pyhwp 의존성으로 자동 설치)
- lxml: HTML 파싱 (pyhwp 의존성으로 자동 설치)
